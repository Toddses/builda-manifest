# builda-manifest

> Simple
> Easy
> Elegant

## Usage

```sh
$ manifest .
```

The manifest will be output to `manifest.json`, in the format

```json
{
	"files": [
		{
			"filename": "img/image1.png",
			"revision": "2018-01-30",
			"filesize": 12345
		},
		{
			"filename": "img/image2.png",
			"revision": "2018-01-30",
			"filesize": 54321
		}
	]
}
```

Super fun.
